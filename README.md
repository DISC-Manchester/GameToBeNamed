# DISCout (SquareSmash)
 DISCout is a .net 7 clone of the old game Brick Breaker.

### CONTROLS
  - **Space** - game start
  - **Enter** - game restart after death
  - **A** - move left
  - **D** - move right
  **DEBUG:**
  - **f1** - instant death
